# Security Policy

RackHead Security is a defensive and educational project.

## Scope

This repository contains documentation, safe toy examples, defensive templates, and early-stage tooling. It is not a production security guarantee.

## Reporting security issues

Please open a private security advisory when GitHub enables it for this repository, or open a minimal public issue that does not include exploitable details.

Do not publish active exploit chains, credentials, malware, stealth/persistence techniques, or instructions for unauthorized access.

## Safe contribution rules

- Use synthetic data.
- Keep examples controlled and local.
- Prefer defensive mitigations over exploit detail.
- Clearly mark limitations.
- Follow responsible disclosure norms for real systems.
